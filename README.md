
# AlphaFold 3 / LLM architecture visualization

Interactive 3D visualization of transformer-style networks and AlphaFold 3–style
architecture, built with Next.js and WebGPU-oriented rendering.

## Running locally

1. Install dependencies: `yarn`
2. Start the dev server: `yarn dev` (serves on port 3002)

The app entry route is `/llm` (the site root redirects there).
